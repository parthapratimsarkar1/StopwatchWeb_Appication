# StopwatchWeb_Appication

## Project Description

This project is a **Stopwatch Web Application** built using **HTML**, **CSS**, and **JavaScript**. It features functionality to start, pause, reset the stopwatch, and record lap times. The stopwatch is interactive and user-friendly with an intuitive layout.

## Features

- **Start**: Start the stopwatch.
- **Pause**: Pause the stopwatch while keeping the current time displayed.
- **Reset**: Reset the stopwatch back to 00:00:00 and clear any lap times.
- **Lap**: Record the current time as a lap.

## Project Structure

1. **index.html**: Contains the structure of the stopwatch.
2. **styles.css**: Provides the design and styling for the stopwatch.
3. **script.js**: Implements the core stopwatch functionalities (start, pause, reset, and lap recording).

## How to Run the Application

1. **Download or clone** the project files to your local machine.
2. Ensure the following files are in the same directory:
    - `index.html`
    - `styles.css`
    - `script.js`
3. Open the `index.html` file in a web browser to run the stopwatch application.

## Instructions

- Click the **Start** button to begin the stopwatch.
- Click the **Pause** button to temporarily stop the timer.
- Click the **Reset** button to reset the timer to 00:00:00.
- Click the **Lap** button to record the current elapsed time as a lap.
- Laps will be displayed in a list below the controls.

## Technologies Used

- **HTML**: Structuring the stopwatch elements.
- **CSS**: Adding styles and improving the UI.
- **JavaScript**: Implementing the core functionality of the stopwatch.
